# Frontend

## Local Development

### Setup

1. Install [Node.js 20.x.x](https://nodejs.org/en)
2. Clone the repository locally.
    ```shell
    git clone https://github.com/principle105/wordpractice.io
    ```
3. Install the dependencies.
    ```shell
    npm install
    ```

### Usage

To start the development server, run

```shell
npm run dev
```
